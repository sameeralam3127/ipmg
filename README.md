**IPMG** is a modern, enterprise-ready network scanning and IP monitoring tool built in Python.

It supports:

- Subnet auto-discovery
- Parallel ping scanning
- Hostname resolution
- XLSX / CSV / JSON reporting
- Scheduled recurring scans
- CLI-based automation

---

## Quick Install

```bash
pip install ipmg
```

Verify:

```bash
ipmg --help
```
